# 常用命令汇总
## 一、Git常用命令
### 1. 基础操作
- git init：初始化Git仓库
- git clone 仓库地址：克隆远程仓库到本地
- git add .：将所有修改添加到暂存区
- git commit -m "提交信息"：提交暂存区代码到本地仓库
- git push：将本地仓库代码推送到远程仓库
- git pull：拉取远程仓库最新代码到本地

### 2. 分支操作
- git branch：查看所有本地分支
- git branch 分支名：创建本地分支
- git checkout 分支名：切换到指定分支
- git checkout -b 分支名：创建并切换到新分支
- git merge 分支名：将指定分支合并到当前分支
- git branch -d 分支名：删除本地分支
- git push origin --delete 分支名：删除远程分支

### 3. 其他常用
- git status：查看当前代码修改状态
- git log：查看提交记录
- git reset --hard 提交ID：回滚到指定提交版本
- git stash：暂存当前修改（未提交）
- git stash pop：恢复暂存的修改

## 二、前端常用命令（Vue3+TS）
### 1. 项目操作
- npm install：安装前端依赖（package.json中配置）
- npm run dev：启动本地开发服务器（默认端口8080）
- npm run build：打包前端项目（生成dist目录，用于部署）
- npm run lint：检查并修复代码格式错误（ESLint+Prettier）

### 2. 依赖操作
- npm install 依赖名 --save：安装生产环境依赖
- npm install 依赖名 --save-dev：安装开发环境依赖
- npm uninstall 依赖名：卸载依赖
- npm update 依赖名：更新依赖版本

### 3. 其他常用
- npm run serve：同npm run dev（部分项目配置）
- npx vue create 项目名：创建新的Vue项目
- npm run test：运行前端测试用例

## 三、后端常用命令（Spring Boot+Java）
### 1. 项目操作（Maven）
- mvn clean：清理项目编译文件（target目录）
- mvn compile：编译项目代码
- mvn package：打包项目（生成jar/war文件，位于target目录）
- mvn spring-boot:run：启动后端服务（本地开发）
- mvn test：运行后端测试用例

### 2. 依赖操作
- mvn dependency:tree：查看项目依赖树（排查依赖冲突）
- mvn clean install：清理、编译、打包，并将jar包安装到本地仓库

### 3. 服务运行（jar包）
- java -jar 项目jar包名.jar：启动后端服务（默认配置）
- java -jar 项目jar包名.jar --spring.profiles.active=dev：指定开发环境启动
- java -jar 项目jar包名.jar --server.port=8081：指定端口启动

## 四、脚本/系统常用命令
### 1. Unix/macOS/Git Bash
- cd 路径：进入指定目录
- ls：查看当前目录文件
- ls -la：查看当前目录所有文件（含隐藏文件、权限）
- mkdir 目录名：创建目录
- rm -rf 目录/文件：删除目录/文件（慎用）
- ln -s 源路径 目标路径：创建符号链接
- chmod +x 脚本名.sh：给脚本添加执行权限
- ./脚本名.sh：运行shell脚本

### 2. Windows（PowerShell/CMD）
- cd 路径：进入指定目录
- dir：查看当前目录文件
- mkdir 目录名：创建目录
- del 文件：删除文件
- rmdir /s 目录名：删除目录（含子目录）
- mklink 目标路径 源路径：创建文件符号链接
- mklink /D 目标路径 源路径：创建目录符号链接
- 脚本名.bat：运行批处理脚本（需管理员权限）
